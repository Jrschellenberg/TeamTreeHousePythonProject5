## Requirements
  ```bash
$ python --version
    Python 3.6.9 
```

## Getting Started (Mac OS / Linux)

- `python3 -m venv p5`
- `source p5/bin/activate`
- `pip install -r requirements.txt`
- `python app.py`


## Getting Started (Windows)
- `python3 -m venv p5`
- `p5\Scripts\activate`
- `pip install -r requirements.txt`
- `python app.py`
